- Hover transitions: 0.3s ease for colors, 0.4s ease for transforms

- Desktop: 1400px+ (large screens)

- Mobile: < 767px

- Reduce font sizes by 20-30%
- Reduce padding/margins by 40%
- Full-width buttons
